# Weather App

A simple, animated weather app built with vanilla HTML, CSS, and JavaScript. Created by Khushi as part of Prodigy Infotech Task 05.

## Features

- Search for any city worldwide (with Indian cities prioritized in results)
- Use current location via geolocation
- Live weather data from Open-Meteo
- Animated scene that changes with the weather (sunny, cloudy, rainy, night)
- Displays temperature, feels-like, humidity, and wind speed

## Files

- `index.html` — page structure/markup
- `style.css` — all styling and animations
- `script.js` — app logic (search, geolocation, weather fetching, rendering)

## Usage

Just open `index.html` in a browser. No build step or dependencies required.

## APIs Used

- [Open-Meteo Geocoding API](https://open-meteo.com/) — city search
- [Open-Meteo Forecast API](https://open-meteo.com/) — weather data
- [BigDataCloud Reverse Geocoding](https://www.bigdatacloud.com/) — location name lookup
